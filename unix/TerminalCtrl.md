光标操作，实用
```
Ctrl+a  光标移动到开始位置
Ctrl+e  光标移动到最末尾
```
删除
```
Ctrl+k  删除此处至末尾的所有内容
Ctrl+u  删除此处至开始的所有内容
```
删除单个
```
Ctrl+d  删除当前字符
Ctrl+h  删除当前字符前一个字符
Ctrl+w  删除此处到左边的单词
```
粘贴，恢复
```
Ctrl+y  粘贴由Ctrl+u， Ctrl+d， Ctrl+w删除的单词
Ctrl+&  恢复 ctrl+h 或者 ctrl+d 或者 ctrl+w 删除的内容
```
清屏,实用
```
Ctrl+l  相当于clear，即清屏
```

移动，实用
```
Ctrl+b  向回移动光标
Ctrl+f  向前移动光标
```
终止命令，实用
```
Ctrl+c  终止进程/命令
```
关闭终端，实用
```
Ctrl+d  关闭终端
```
全屏终端
```
F11 全屏切换
```
开启新的终端，实用
```
Ctrl+Alt+T
```
